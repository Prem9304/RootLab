# RootLab

RootLab is an advanced web-based ethical hacking platform and virtual cyber range. It provides a safe, contained environment for cybersecurity practice, featuring a suite of integrated tools, a Kali Linux virtual sandbox, and an AI-assisted learning curriculum.

## Features

- **Integrated Security Tools:** Run powerful web-based UI wrappers around industry-standard tools like Nmap, Nikto, Katana, theHarvester, and WRK.
- **Nessus Integration:** Built-in REST API integration to connect and drive Tenable Nessus scans directly from the RootLab interface.
- **Kali Linux Sandbox:** A fully functional, web-accessible terminal connected to a secure Dockerized Kali Linux container with pre-installed penetration testing toolkits.
- **AI-Powered Learning:** A dynamic learning dashboard with curriculum paths, AI-driven guidance, and real-time vulnerability analysis.
- **Dynamic Workspaces:** A modern, customizable desktop-like UI (window manager, taskbar) to manage multiple tools concurrently.

## Getting Started

1. Clone the repository.
2. Ensure you have Docker and Node.js installed.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser to `http://localhost:3000`.
